# Smart City Portal

A smart city web portal showcasing integrated city services including traffic control, hospital management, ambulance tracking, and public information dashboards.

## Features

- Interactive smart city overview with animated visuals
- Traffic control monitoring section
- Hospital and healthcare information
- Ambulance tracking UI
- Live camera feed integration section
- Responsive design with CSS animations
- Project synopsis documentation included

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Assets:** Custom images, video feeds, GIF animations

## Getting Started

### Prerequisites

- Any modern web browser

### Running

Simply open `index.html` in your browser:

```bash
git clone https://github.com/amanshakya2001/smart-city-portal-project.git
cd smart-city-portal-project
# Open index.html in browser
start index.html
```

Or serve with a local server:

```bash
npx serve .
```
